# 📈 Επενδυτικό Dashboard

Βοηθός απόφασης για μετοχές & ETF: live τιμές, γραφήματα, τεχνικοί/θεμελιώδεις
δείκτες, σκορ ανά μετοχή και ιδέα κατανομής κεφαλαίου.

> ⚠️ **Δεν είναι επίσημη επενδυτική συμβουλή.** Οι επενδύσεις έχουν κίνδυνο
> απώλειας κεφαλαίου. Το εργαλείο βοηθάει να αποφασίζεις — δεν προβλέπει το μέλλον.
> Δεν συνδέεται με DeGiro· τις εντολές τις περνάς εσύ χειροκίνητα.

---

## ▶️ Πώς το τρέχω στον υπολογιστή (Windows)

1. Διπλό-κλικ στο **`Εκκίνηση.bat`**.
2. Ανοίγει μόνο του ο browser στο `http://localhost:8501`.
3. Για να το σταματήσεις, κλείσε το μαύρο παράθυρο.

Πρώτη φορά μόνο, αν δεν έχουν εγκατασταθεί οι βιβλιοθήκες, τρέξε μία φορά:
```
py -m pip install -r requirements.txt
```

## 📱 Πώς το βλέπω από το ΚΙΝΗΤΟ

### Επιλογή Α — δωρεάν online (προτεινόμενη, το βλέπεις από παντού)
1. Φτιάξε δωρεάν λογαριασμό στο GitHub και ανέβασε αυτόν τον φάκελο.
2. Πήγαινε στο **share.streamlit.io** → *New app* → διάλεξε το repo και το `app.py`.
3. Σου δίνει ένα link (π.χ. `https://onoma.streamlit.app`) — άνοιξέ το στο κινητό
   και βάλ' το στην αρχική οθόνη σαν εφαρμογή.

### Επιλογή Β — στο ίδιο WiFi (γρήγορο, χωρίς hosting)
1. Τρέξε στον υπολογιστή:
   ```
   py -m streamlit run app.py --server.address 0.0.0.0
   ```
2. Βρες την IP του PC (`ipconfig` → IPv4, π.χ. `192.168.1.10`).
3. Στο κινητό (ίδιο WiFi) άνοιξε: `http://192.168.1.10:8501`.

## 🧭 Πώς το χρησιμοποιώ

- **Watchlist** (πλαϊνό μενού): βάλε σύμβολα Yahoo Finance χωρισμένα με κόμμα.
  - Παραδείγματα: `AAPL` (Apple), `MSFT`, `NVDA`, `ASML.AS` (Άμστερνταμ),
    `VUAA.DE` (S&P 500 ETF σε €), `IWDA.AS` (MSCI World ETF).
- **Επισκόπηση & Κατανομή**: κατάταξη με σκορ + ιδέα πώς να μοιράσεις τα 40.000€.
- **Ανάλυση μετοχής**: candlestick + MA50/MA200, RSI, MACD και εξήγηση του σκορ.

### Πώς βγαίνει το σκορ (0–100, διαφανές)
- Τάση (τιμή vs κινητοί μέσοι 50/200)
- RSI (υπεραγορασμένο/υπερπουλημένο)
- MACD (ορμή)
- Απόδοση 6μήνου

Σκορ ≥ 65 = «Ισχυρό», 45–64 = «Ουδέτερο», < 45 = «Αδύναμο».
**Ένδειξη, όχι βεβαιότητα.**

## 🧾 Φορολογικό φίλτρο (ιδιώτης, φορολογικός κάτοικος Ελλάδας)

Στο πλαϊνό μενού διάλεξε τι θέλεις να βλέπεις:
- **Μόνο πλήρως αφορολόγητα** (🟢): accumulating UCITS ETF (χωρίς μέρισμα) ή
  μετοχές χωρίς μέρισμα — μηδενικός φόρος (υπεραξία 0% + κανένα μέρισμα).
- **Μόνο αφορολόγητη υπεραξία** (🟢+🟡): + όσα έχουν αφορολόγητο κέρδος πώλησης
  αλλά μέρισμα 5%.

Βάση: **Ν.4172/2013 άρθρα 42-43**. Για ιδιώτη με κατοχή < 0,5% εισηγμένης
εταιρείας, η **υπεραξία μετοχών είναι αφορολόγητη**· τα **μερίσματα** φορολογούνται
**5%**· τα **UCITS ETF** έχουν αφορολόγητη υπεραξία (τα accumulating δεν δίνουν
μέρισμα → 0% συνολικά). Αμερικανικά (US) ETF συνήθως δεν είναι διαθέσιμα σε EU
retail (PRIIPs) — ψάξε το UCITS αντίστοιχο (κατάληξη `.DE`, `.AS`, `.L`).

> ⚠️ Ενδεικτικός χαρακτηρισμός, **όχι φορολογική συμβουλή**. Η ερμηνεία της ΑΑΔΕ
> εξελίσσεται — επιβεβαίωσε με φοροτεχνικό πριν επενδύσεις.

**Παραδείγματα πλήρως αφορολόγητων accumulating UCITS ETF:**
`VUAA.DE` (S&P 500), `SXR8.DE` (S&P 500), `IWDA.AS` (MSCI World), `EUNL.DE` (MSCI World).

## 💡 Βασικοί κανόνες ρίσκου (για τα 40.000€)
- Μην βάζεις πάνω από ~20–25% σε μία μόνο μετοχή.
- Κράτα ένα μέρος σε ETF ευρείας αγοράς για σταθερότητα.
- Σκέψου ορίζοντα χρόνου: το «1 χρόνο» είναι μικρό διάστημα — οι διακυμάνσεις
  μεγάλες. Όσο μεγαλύτερος ο ορίζοντας, τόσο μικρότερο το ρίσκο κακής χρονιάς.
